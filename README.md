# Vimfiles

My humble vimfiles. Using nightly neovim. Install with

``` shell
git clone git@github.com:rafadc/vimfiles.git .vim
ln -s ~/.vim ~/.config/nvim
```

The in your vim run

```
:PlugInstall
```

And you are ready to go!
